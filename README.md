# Saudi Weather Discover App

A small single-page app that lets users enter a Saudi city and see the current weather in Celsius or Fahrenheit.

## How to use

1. Open `index.html` in your browser.
2. Type a city name like `Riyadh`, `Jeddah`, or `Dammam`.
3. Click **Search weather**.
4. Toggle the unit between Celsius and Fahrenheit using the button.

## Notes

- The app uses the free Open-Meteo geocoding and weather API.
- No API key is required.
